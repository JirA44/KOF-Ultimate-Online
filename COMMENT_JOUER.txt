🎮 MANETTE NE FONCTIONNE PAS?
════════════════════════════════════════════════════════════════

Solution: Utiliser le MAPPER!

1. Double-cliquez sur: START_KOF_ULTIMATE.bat
2. Choisissez option [6] - Mapper Manette → Clavier
3. Une fenêtre s'ouvre qui convertit votre manette en clavier
4. Laissez cette fenêtre ouverte
5. Lancez le jeu (option 1, 2 ou 3)
6. Jouez avec votre manette!

Le mapper convertit:
  • D-Pad/Stick → Flèches directionnelles
  • A → S (coup léger)
  • B → D (coup moyen)
  • X → F (coup fort)
  • Y → G (spécial)
  • Start → Entrée
  • Back → Échap
