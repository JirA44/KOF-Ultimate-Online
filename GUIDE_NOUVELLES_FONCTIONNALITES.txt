🎨 SYSTÈME DE PROFILS
═══════════════════════════════════════════════════════════════════════

📋 FONCTIONNALITÉS:
   • Créer plusieurs profils utilisateur
   • Choisir un fond animé pour chaque profil
   • Basculer entre profils facilement
   • Sauvegarde automatique des préférences

🎨 FONDS DISPONIBLES:

1. KOF DM BLACK (Default)
   → Fond par défaut du jeu
   → Animations: 6 layers

2. Flammes Bleues
   → Fond avec flammes bleues animées
   → Animations: 3 layers (plus léger)

3. Énergie Rouge
   → Fond avec énergie rouge
   → Animations: 3 layers

4. Minimaliste
   → Fond simple et épuré
   → Animations: 1 layer (performance maximale)

5. Ultra Animé
   → Maximum d'animations
   → Animations: 8 layers (effet spectaculaire)

🚀 COMMENT UTILISER:

Option A - Via le launcher:
   1. Double-cliquez sur LAUNCHER_COMPLET.bat
   2. Choisissez "2. Gestionnaire de profils"
   3. Sélectionnez un fond animé
   4. Le fond sera appliqué au prochain lancement

Option B - Directement:
   1. Lancez: python PROFILE_MANAGER.py
   2. Menu interactif avec toutes les options

📝 UTILISATION DÉTAILLÉE:

Menu principal:
   1. Changer de profil → Basculer entre vos profils
   2. Créer nouveau profil → Ajouter un profil
   3. Changer le fond → Sélectionner parmi les 5 fonds
   4. Lister fonds → Voir tous les fonds disponibles
   5. Lancer le jeu → Démarrer avec le profil actif

💾 FICHIERS CRÉÉS:
   • profiles.json → Sauvegarde de vos profils
   • data/system.def.backup_profile → Backup de l'original

🧪 TESTS AUTOMATIQUES SANS CLAVIER
═══════════════════════════════════════════════════════════════════════

⚠️  IMPORTANT:
   Ces tests envoient les touches DIRECTEMENT à la fenêtre du jeu
   via l'API Windows (PostMessage).

   ✅ Votre clavier reste TOTALEMENT LIBRE
   ✅ Vous pouvez taper pendant les tests
   ✅ Aucune interférence avec vos touches

🎮 FONCTIONNALITÉS:
   • Lance le jeu automatiquement
   • Navigue dans les menus
   • Sélectionne des personnages
   • Lance des combats
   • Analyse les erreurs

🚀 COMMENT UTILISER:

Option A - Via le launcher:
   1. Double-cliquez sur LAUNCHER_COMPLET.bat
   2. Choisissez "3" pour 1 test
   3. Ou "4" pour 5 tests continus

Option B - Directement:
   python TEST_AUTO_SANS_CLAVIER.py

📊 MODES DISPONIBLES:
   1. Test unique → 1 cycle complet (~30s)
   2. Tests continus → 5 cycles (~3 minutes)

🔍 CE QUE FONT LES TESTS:
   1. Lancent KOF_Ultimate_Online.exe
   2. Trouvent la fenêtre du jeu
   3. Passent l'écran titre (Entrée)
   4. Sélectionnent mode Arcade
   5. Naviguent dans la grille de personnages
   6. Sélectionnent 2 personnages
   7. Observent le match 15 secondes
   8. Quittent et ferment le jeu
   9. Analysent mugen.log pour erreurs

💡 DIFFÉRENCE AVEC LES ANCIENS TESTS:

   ❌ ANCIENS (pyautogui):
      • Utilisaient votre clavier
      • Écrivaient n'importe quoi si vous tapiez
      • Perturbaient votre travail

   ✅ NOUVEAUX (win32api):
      • Messages directs à la fenêtre
      • Zéro impact sur votre clavier
      • Vous pouvez travailler pendant les tests

❓ FAQ
═══════════════════════════════════════════════════════════════════════

Q: Le fond ne change pas ?
A: Relancez le jeu après avoir changé le fond dans les profils.

Q: Les tests perturbent mon clavier ?
A: NON ! Les nouveaux tests (TEST_AUTO_SANS_CLAVIER.py) utilisent
   l'API Windows et n'affectent PAS votre clavier.

Q: Comment revenir au fond par défaut ?
A: Gestionnaire profils → Choix 3 → Sélectionner "KOF DM BLACK"

Q: Puis-je créer mes propres fonds ?
A: Oui ! Éditez PROFILE_MANAGER.py et ajoutez vos animations
   dans le dictionnaire self.backgrounds

Q: Les profils sont-ils sauvegardés ?
A: Oui, dans profiles.json. Backupez ce fichier pour conserver
   vos profils.
