✅ SERVEUR DE MATCHMAKING MMR
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📁 matchmaking_server.py
   • Serveur TCP multi-threading
   • Port: 9999
   • Système MMR adaptatif
   • Files d'attente par mode (Ranked/Quick/Custom)
   • Matchmaking automatique
   • Notifications temps réel
   • Stats du serveur en direct

Fonctionnalités:
   ✓ Connexion multi-joueurs
   ✓ File d'attente ranked (MMR strict)
   ✓ File d'attente rapide (MMR flexible)
   ✓ Matchmaking basé sur:
     - Différence MMR (50 + 5 par seconde d'attente)
     - Temps d'attente
   ✓ Création et notification de matchs
   ✓ Gestion des déconnexions

✅ CLIENT DE MATCHMAKING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📁 matchmaking_client.py
   • Client de connexion au serveur
   • Communication socket TCP
   • Système de callbacks
   • Gestion asynchrone des messages
   • Thread de réception

Méthodes:
   ✓ connect()       - Connexion au serveur
   ✓ search_match()  - Recherche de partie
   ✓ cancel_search() - Annulation
   ✓ disconnect()    - Déconnexion

Callbacks:
   ✓ on_match_found    - Match trouvé
   ✓ on_connected      - Connexion établie
   ✓ on_disconnected   - Déconnexion
   ✓ on_error          - Erreur
